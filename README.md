# Affiliate Agent Stack

A multi-agent system designed to build, manage, and optimize an affiliate marketing website.

## 🤖 The Agents
*   **CEO**: Strategy and oversight.
*   **Portfolio Manager**: Product selection and market analysis.
*   **Data Analyst**: Performance tracking and SEO optimization.
*   **Product Manager**: Feature definition and roadmap.
*   **Developers**: Implementation of the site and integrations.

## 🛠 Tech Stack
*   **AI**: OpenRouter (LLMs).
*   **Data**: SearchAPI.
*   **Frontend**: Next.js.
*   **Backend**: Python.

## 🚀 Getting Started

### Prerequisites
*   Node.js & npm
*   Python 3.10+
*   Git

### Setup
1.  Clone the repository.
2.  Install dependencies (instructions to follow).
3.  Set up environment variables:
    *   `OPENROUTER_API_KEY`
    *   `SEARCHAPI_KEY`
